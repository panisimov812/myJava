# База знаний itmoLessons

Локальная страница для просмотра конспектов и кода из пакета `itmoLessons`.

## Страницы

- **По урокам** (`lessons.html`) — обзор по каждому уроку: сразу видна основная информация (все .txt), примеры кода с превью и ссылкой «Открыть полностью». Удобно для повторения и быстрого доступа к теории.
- **Все файлы** (`index.html`) — полный список файлов по урокам, поиск по содержимому, просмотр любого файла целиком.

## Как открыть

1. **Обновить данные** (после изменений в `src/main/java/itmoLessons/`):
   ```bash
   cd /Users/anisimovpk/Desktop/myJava
   python3 build_knowledge_base.py
   ```

2. **Открыть страницу**:
   - Рекомендуется начать с **По урокам**: двойной клик по `lessons.html`.
   - Или в браузере: `file:///.../knowledge-base/lessons.html`
   - Для просмотра всех файлов и поиска: `index.html`. Ссылка «Открыть полностью» на странице уроков ведёт в `index.html` с уже открытым файлом.

Либо поднять локальный сервер: `python3 -m http.server 8000` из корня проекта, затем открыть `http://localhost:8000/knowledge-base/lessons.html` или `.../index.html`.

## Возможности

- **По урокам**: оглавление «Перейти к уроку», по каждому уроку — краткое описание, блок «Основная информация» (все .txt), блок «Примеры кода» (раскрывающиеся превью .java и ссылка на полный просмотр).
- **Все файлы**: боковая панель по урокам, поиск по содержимому, подсветка совпадений при просмотре файла.
- Верхнее меню на обеих страницах: переключение между «По урокам» и «Все файлы».

Данные встроены в HTML, интернет нужен только для шрифтов (JetBrains Mono, Unbounded).
