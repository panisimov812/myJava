 Модификаторы доступа
        private - область видимости текущий в классе
        protected - обращаться внутри текущего класса, в наследниках, внутри паетка
        public - дает возможность обращаться откуда угодно
        default - когда модификатора нет (еще называется package private) обращаться из того же класса или пакета
